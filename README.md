Mini-e-commerce-v2 

Es un proyecto que surge del "Mini-e-commerce" en el cual hice una tiendo virtual para un negocio de mi ciudad,
el objetivo de este versionado 2 es agregarle la interacción de un backend y una base de datos ya que inicialmente este proyecto solo era frontend
y no había una carga de datos desde una base de datos sino que estaban cargados estáticamente. Esta mejora la hice aparte para poder comparar los resultados
de cada versión y poder tener un feedback automático.
Las ventajas de estas mejoras principalmente fueron que me permitió practicar tanto Node.js, Express.js y PostgreSQL así como también la lógica de la parte backend y fullstack.

Prácticas que agregué:
  - Pude aplicar en un proyecto "real"
      - Async and await.
      - Promises.
      - Peticiones AJAX (fetchs).
      - Server.
      - DB.
      - Peticiones querys.
  - División de un proyecto en frontend, backend y base de datos.
  - Conocimientos en proyectos fullstack.

Potenciales mejoras:
  - Escalabilidad del sistema.
  - Implementar una DB en la nube para las imágenes de los productos.
  - Una sección donde permita al administrador editar, agregar o eliminar productos (básicamente un CRUD). Próxima a trabajar.
  - Una sección de logeo del administrador.
